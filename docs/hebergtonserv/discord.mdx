# Guide pour la Prise de Rôle Client sur Discord

Dans ce guide, nous allons vous montrer comment prendre un rôle client sur un serveur Discord. Assurez-vous de suivre ces étapes pour accéder à certaines fonctionnalités spécifiques du serveur.

## Étape 1 : Rejoindre le Serveur

Tout d'abord, assurez-vous d'avoir rejoint le serveur Discord approprié. Si vous n'êtes pas encore membre du serveur, demandez une invitation à l'administrateur du serveur ou utilisez un lien d'invitation.

## Étape 2 : Accéder aux Rôles

1. Cliquez sur le nom du serveur en haut à gauche de l'interface Discord pour afficher la liste des canaux.
2. Faites défiler la liste jusqu'à ce que vous trouviez la section "Rôles" ou "Roles". Elle peut être située sous une catégorie telle que "Informations du Serveur".
3. Cliquez sur cette section pour voir la liste des rôles disponibles.

![Capture d'écran des rôles](https://i.imgur.com/4974wW2.png)

## Étape 3 : Sélectionner un Rôle

1. Une liste de rôles s'affichera. Cliquez sur le rôle que vous souhaitez prendre. Certains serveurs peuvent nécessiter une réaction à un message ou une autre action pour obtenir le rôle.
2. Une fois que vous avez cliqué sur le rôle, vous devriez voir une confirmation que le rôle vous a été attribué avec succès.

![Capture d'écran de la sélection du rôle](./img/link-client.png)

## Étape 4 : Utiliser le Rôle

Maintenant que vous avez pris le rôle client, vous pouvez accéder aux fonctionnalités spécifiques réservées aux membres de ce rôle sur le serveur Discord. Assurez-vous de respecter les règles du serveur et de participer de manière appropriée.

C'est tout ! Vous avez maintenant pris un rôle client sur Discord. Profitez de votre expérience sur le serveur !
