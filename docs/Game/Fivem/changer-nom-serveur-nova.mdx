Pour changer le nom de votre serveur NovaLife, suivez ces étapes :

1. Accédez à notre panel de gestion de jeux à l'adresse suivante : https://manage.hebergtonserv.fr.
2. Sélectionnez votre serveur NovaLife.
3. Allez dans la section "Configuration".
4. Cliquez sur "Paramètres de démarrage".
5. Modifiez le champ "serverListName" avec le nouveau nom.
6. Redémarrez votre serveur pour appliquer les modifications.
7. Votre serveur NovaLife portera désormais le nouveau nom.
