# Mise à Jour de l'Alpine (FXServer)

Suivez ces étapes pour mettre à jour ou installer la dernière version de l'Alpine pour votre serveur FXServer via le panel `manage.hebergtonserv.com`.

## Accès au Panel de Gestion

1. Allez sur le panel de gestion à l'adresse [manage.hebergtonserv.com](https://manage.hebergtonserv.com).

## Mise à Jour ou Installation de l'Alpine

2. Dans votre serveur, naviguez vers la section **Outils Gestionnaire de Mode**.
   
   ![Gestionnaire de Mode](https://i.imgur.com/pNLp9SB.png)

3. Cherchez la section **Alpine Latest**.

   ![Alpine Latest](https://i.imgur.com/8uGoUKL.png)

4. Pour installer ou mettre à jour l'Alpine :
   - Si **Alpine Latest** n'est pas déjà installé : Cliquez sur **Installer** pour mettre en place la dernière version.
   - Si **Alpine Latest** est déjà installé : 
     - Cliquez d'abord sur **Désinstaller** pour supprimer la version actuelle.
     - Ensuite, cliquez sur **Installer** pour installer la dernière version publiée par FiveM.

   ![Installation Alpine](https://i.imgur.com/p6zsg0y.png)

## Finalisation

Après l'installation ou la mise à jour, votre serveur FXServer sera équipé de la dernière version de l'Alpine. Il est recommandé de redémarrer votre serveur pour que les changements prennent effet.

En suivant ces étapes, vous vous assurez que votre serveur FXServer utilise la version la plus récente et la plus stable d'Alpine, ce qui peut apporter des améliorations de performance et de sécurité.
