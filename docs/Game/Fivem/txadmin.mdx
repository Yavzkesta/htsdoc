# Configuration de TXAdmin sur Hebergtonserv.com

Suivez ces étapes pour configurer TXAdmin après avoir commandé un serveur sur `hebergtonserv.com`.

## Accès au Panel et Configuration Initiale

1. Connectez-vous à votre serveur depuis votre panel préféré sur `hebergtonserv.com`.

2. Vérifiez que l'option TXAdmin est bien paramétrée dans **Configuration -> Paramétrage de Démarrage**.

   ![Paramétrage de Démarrage](https://i.imgur.com/cUZs0bw.png)

3. Activez l'option TXAdmin.

   ![Activer TXAdmin](https://i.imgur.com/zfnh5c7.png)

## Démarrage du Serveur et Configuration de TXAdmin

4. Une fois activé, allez dans la console et démarrez votre serveur.

   ![Démarrage Serveur](https://i.imgur.com/oVWbPvu.png)
   ![Console Serveur](https://i.imgur.com/QPzdwnt.png)

5. Une fois démarré, un lien va s'afficher dans la console avec un code PIN.

   ![Lien TXAdmin](https://i.imgur.com/KKch3fi.png)

6. Accédez au lien, entrez le code PIN, puis cliquez sur "Link Account".

   ![Link Account](https://i.imgur.com/i3OQDgR.png)

7. Autorisez TXAdmin à accéder à votre compte CFX.

   ![Autorisation CFX](https://i.imgur.com/Y1wRJuA.png)

8. Créez votre mot de passe, cochez les TOS (Termes de Service), puis cliquez sur "Create Master Account".

   ![Création Compte](https://i.imgur.com/mWQaPs1.png)

## Configuration Finale de TXAdmin

9. Les prochaines étapes concernent la configuration de TXAdmin, qui sera liée à la racine de votre serveur Hebergtonserv. Cliquez sur "Next".

   ![Configuration Suivante](https://i.imgur.com/H4yR2AH.png)

10. Entrez le nom de votre serveur et cliquez sur "Next".

    ![Nom du Serveur](https://i.imgur.com/MmtaHft.png)

11. Pour "Deployment Type", sélectionnez "Existing Server Data". (Nous verrons dans un autre tutoriel pour une version autopush.)

    ![Type de Déploiement](https://i.imgur.com/JOsdFZl.png)

12. Entrez le chemin physique où se trouve le `server.cfg` (`/home/container/`).

    ![Chemin Serveur](https://i.imgur.com/MdcTdxk.png)

13. TXAdmin va autodétecter votre `server.cfg`. Cliquez sur "Save".

    ![Autodétection server.cfg](https://i.imgur.com/kYrNheS.png)

14. Il ne vous reste plus qu'à appuyer sur le bouton "Save & Start Server".

    ![Démarrer Serveur](https://i.imgur.com/75xMGaS.png)

En suivant ces instructions, vous aurez configuré TXAdmin pour votre serveur FiveM hébergé sur Hebergtonserv. Cette configuration vous permettra de gérer votre serveur plus efficacement et d'accéder à des outils d'administration avancés.
