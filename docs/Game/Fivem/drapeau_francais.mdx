# Vous cherchez à afficher le drapeau français sur le listage de votre serveur FiveM ? Voici un guide simple pour vous aider à configurer votre hébergeur FiveM afin de montrer le drapeau français.

Étapes pour Afficher le Drapeau Français
1. Modifier le Fichier server.cfg :
Ouvrez le fichier server.cfg de votre serveur FiveM et ajoutez ou modifiez la ligne suivante pour définir la langue et la région comme étant la France :
```text
sets locale "fr-FR"
```
2.  Redémarrer le Serveur :
Après avoir enregistré les modifications dans le fichier server.cfg, redémarrez votre serveur FiveM. Cela permettra d'appliquer les nouvelles configurations et d'afficher le drapeau français dans le listage de votre serveur.

Pourquoi Utiliser le drapeau français ?
Utiliser un hébergement FiveM avec une configuration de localisation précise améliore non seulement la visibilité de votre serveur dans les recherches locales mais aussi l'expérience utilisateur pour les joueurs francophones. Un serveur correctement localisé avec le drapeau français peut attirer plus de joueurs français, augmentant ainsi l'engagement et la popularité de votre serveur.

Conclusion
En suivant ces étapes simples, vous pouvez facilement configurer votre serveur FiveM pour afficher le drapeau français. Assurez-vous de redémarrer votre serveur après chaque modification pour voir les changements prendre effet.
