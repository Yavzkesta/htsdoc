# Connexion à PHPMyAdmin avec une chaîne de connexion MySQL mise à jour

Cette page fournit des instructions sur la manière de se connecter à PHPMyAdmin en utilisant une chaîne de connexion MySQL mise à jour, compatible avec `oxmysql` et `mysql-async`.

## Chaîne de Connexion Obsolète

La chaîne de connexion suivante est maintenant obsolète pour `oxmysql` :

```text
set mysql_connection_string "server=IP_PHP_SANS_PORTS;database=NOM;userid=NOM_UTILISATEUR_PHP;password=MOT_DE_PASSE_PHP"
```

Ce format peut ne pas fonctionner avec les dernières versions de `oxmysql` et devrait être mis à jour.

## Chaîne de Connexion Mise à Jour

Pour la compatibilité avec `oxmysql` et `mysql-async`, utilisez le format de chaîne de connexion suivant :

```text
set mysql_connection_string "mysql://NOM_UTILISATEUR_PHP:MOT_DE_PASSE_PHP@IP_PHP_SANS_PORTS/NOM?charset=utf8mb4"
```

Remplacez `NOM_UTILISATEUR_PHP` par votre nom d'utilisateur MySQL, `MOT_DE_PASSE_PHP` par votre mot de passe MySQL (assurez-vous qu'il ne contient pas de '@'), `IP_PHP_SANS_PORTS` par l'adresse IP de votre serveur MySQL, et `NOM` par le nom de votre base de données.
