# Création d'une Base de Données sur Hebergtonserv

Ce guide explique comment créer une base de données pour votre serveur sur `hebergtonserv.com`.

## Accès à la Gestion des Bases de Données

1. Connectez-vous à votre panel sur `hebergtonserv.com`.

2. Naviguez vers **Gestion -> Bases de données**.

   ![Gestion Bases de Données](https://i.imgur.com/hA3snXS.png)

## Création de la Base de Données

3. Cliquez sur le bouton **Créer** pour commencer la création d'une nouvelle base de données.

   ![Créer Base de Données](https://i.imgur.com/72QsStd.png)

4. Sélectionnez un hôte pour votre base de données.

5. Entrez un nom pour votre base de données. Laissez la troisième case en "%". Cette configuration est cruciale car elle définit les autorisations de connexion à votre base de données.

   ![Configuration Base de Données](https://i.imgur.com/qTP5wWk.png)

6. Une fois les informations saisies, confirmez la création de votre base de données.

## Confirmation de la Création

7. Après avoir créé la base de données, elle devrait apparaître dans votre liste de bases de données sur le panel.

   ![Base de Données Créée](https://i.imgur.com/jh6rkHs.png)

Votre base de données est maintenant créée et prête à être utilisée pour votre serveur. Assurez-vous de noter les informations de connexion, car elles seront nécessaires pour configurer votre serveur ou vos applications qui nécessitent un accès à cette base de données.
