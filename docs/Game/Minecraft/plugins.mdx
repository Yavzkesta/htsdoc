# Comment ajouter un plugin sur sont serveur minecraft ?
    :::danger Take care
    Il faut absolument un serveur compatible avec les plugins (Par exemple Spigot,Paper,Mohist...)

    Il faut ensuite vous rendre dans votre SFTP (tous est disponible dans Details SFTP sur votre panel), puis mettre votre plugin dans le dossier Plugins puis redémarrer votre serveur.
    Quand cela est fais je vous invite à vérifier si le plugin c'est correctement chargé, pour cela nous allons faire la commande plugins sur la console, si le plugin est en vert il est chargé, si il est en rouge il y as eu un soucis dans le chargement du plugins il faudra donc regarder la console à recherche du message du plugins indiquant que celui-ci n'as pas charger.
