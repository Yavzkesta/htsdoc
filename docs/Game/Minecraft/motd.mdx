:::tip My tip

Le motd de Minecraft est un message qui apparaît dans la liste des serveurs lorsque les joueurs se connectent.

:::

Pour modifier le motd via le fichier server.properties, vous devez suivre ces étapes:

Ouvrez le fichier server.properties avec un éditeur de texte.

Recherchez la ligne commençant par motd=.

Modifiez cette ligne pour pointer vers le nouveau motd. Par exemple, si votre motd est "Bienvenue sur notre serveur!", vous pouvez modifier la ligne en motd=Bienvenue sur notre serveur!.

Sauvegardez les modifications apportées au fichier et fermez-le.

Redémarrez votre serveur Minecraft.

Maintenant, lorsque les joueurs se connectent à votre serveur, le nouveau motd apparaîtra dans la zone d'accueil.
