# Creating a Database on Hebergtonserv

This guide explains how to create a database for your server on `hebergtonserv.com`.

## Accessing the Database Management

1. Log in to your panel on `hebergtonserv.com`.

2. Navigate to **Management -> Databases**.

   ![Database Management](https://i.imgur.com/hA3snXS.png)

## Creating the Database

3. Click on the **Create** button to start setting up a new database.

   ![Create Database](https://i.imgur.com/72QsStd.png)

4. Select a host for your database.

5. Enter a name for your database. Leave the third box as "%". This configuration is crucial as it defines the connection permissions for your database.

   ![Database Configuration](https://i.imgur.com/qTP5wWk.png)

6. Once you have entered the information, confirm the creation of your database.

## Confirmation of Creation

7. After creating the database, it should appear in your list of databases on the panel.

   ![Database Created](https://i.imgur.com/jh6rkHs.png)

Your database is now created and ready to be used for your server. Be sure to note the connection information, as it will be necessary to configure your server or applications that require access to this database.
