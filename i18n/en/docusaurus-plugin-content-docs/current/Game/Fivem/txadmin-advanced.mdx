# Configuring TXAdmin with Push Server on Hebergtonserv.com

This guide outlines the steps for configuring TXAdmin using the "Push Server" method on a server ordered through `hebergtonserv.com`.

## Initial Setup of TXAdmin

1. Log in to your server from your panel on `hebergtonserv.com`.

2. Ensure that the TXAdmin option is properly set in **Configuration -> Startup Settings**.

   ![Startup Settings](https://i.imgur.com/cUZs0bw.png)

3. Activate the TXAdmin option.

   ![Activate TXAdmin](https://i.imgur.com/zfnh5c7.png)

4. Start your server from the console.

   ![Server Start](https://i.imgur.com/oVWbPvu.png)
   ![Server Console](https://i.imgur.com/QPzdwnt.png)

5. A link and a PIN code will be displayed in the console.

   ![TXAdmin Link](https://i.imgur.com/KKch3fi.png)

6. Follow the link, enter the PIN, and click on "Link Account".

   ![Link Account](https://i.imgur.com/i3OQDgR.png)

7. Authorize TXAdmin to access your CFX account.

   ![Authorize CFX](https://i.imgur.com/Y1wRJuA.png)

8. Create your password, check the TOS, and then click on "Create Master Account".

   ![Create Account](https://i.imgur.com/mWQaPs1.png)

## Deployment of the Recipe

9. Click on "Next" to start the deployment.

   ![Start Deployment](https://i.imgur.com/H4yR2AH.png)

10. Enter the name of your server and click on "Next".

    ![Server Name](https://i.imgur.com/MmtaHft.png)

11. For "Deployment Type", select "Popular Recipes RECOMMENDED".

    ![Deployment Type](https://i.imgur.com/dyb8qim.png)

12. Select the desired template, for example, "esx legacy".

    ![Select Template](https://i.imgur.com/s7cDiQl.png)

13. Clean up the root of your server by deleting the `resource` folder and `server.cfg`.

    ![Root Cleanup](https://i.imgur.com/d1g73P1.png)

14. Return to TXAdmin to complete the deployment. Click on "Save" for the "Data location" step.

    ![Save Data Location](https://i.imgur.com/IyWElyF.png)

15. Click on "Go to recipe deployer" and follow the steps.

    ![Recipe Deployer](https://i.imgur.com/yd0u66O.png)

16. Enter your CFX license and set up your database according to the [database creation tutorial](/Game/Fivem/create-BDD).

    ![Database Configuration](https://i.imgur.com/6tsPLl2.png)

17. Click on "Run Recipe".

    ![Run Recipe](https://i.imgur.com/XvZdNj9.png)

18. Once the deployment is complete, click on "Save & Run Server".

    ![Save and Start Server](https://i.imgur.com/ksqdVVu.png)

## Final Cleanup and Startup

19. Stop your server from the Hebergtonserv panel.

    ![Stop Server](https://i.imgur.com/4fRuCdk.png)

20. Move the necessary files from `txData` to the root of your server.

    ![Move Files](https://i.imgur.com/z6LBFJP.png)

21. Delete the unnecessary folder in `txData`.

    ![Delete Folder](https://i.imgur.com/j7Jpzgv.gif)

22. Restart your server and access your TXAdmin panel to finalize the setup.

    ![Restart Server](https://i.imgur.com/ak1ISF6.gif)

Your server is now ready and configured with TXAdmin using the "Push Server" method.
